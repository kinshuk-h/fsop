\doxysubsection{Detailed Description}
Exposes abstractions over system calls related to the filesystem. 

This namespace exposes classes for \mbox{\hyperlink{structfsop_1_1_file}{File}} I/O as well as functions for performing actions corresponding to various system calls such as opening files and pipes, inspecting files (via stat), chaninging directories as well as utility functions for mapping arguments to types accepted by the system calls.
