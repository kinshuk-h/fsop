\doxysubsection{Detailed Description}
Represents a base type to store argument description and parsing information. 

\mbox{\hyperlink{structargparse_1_1_argument}{Argument}} objects maintain information about command-\/line arguments for effective parsing. Further, the provided data aids in providing a concise description to the user.
