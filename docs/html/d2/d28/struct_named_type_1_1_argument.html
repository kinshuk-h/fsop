Detailed Description

template<typename ValueType, typename Descriptor>
struct NamedType< ValueType, Descriptor >::Argument

Type for representing variables usable for named arguments in functions.

Variables of this type upon assignment returns a NamedType object, which allows for syntax similar to the following:

using Size = NamedType<int, struct SizeTag>;
Size::Argument size;
void init_window(Size size);
...
init_window(size = 100);
NamedType
Generates Named Types for use in arguments of functions to better elaborate the context.
