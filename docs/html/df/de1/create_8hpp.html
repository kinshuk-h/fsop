void fsop::create_file ( std::string_view  path,
mode_t  permissions,
bool  overwrite = false 
)

Creates a new regular file in the filesystem.

For successful operation, ensure that the file does not previously exist, all directories along the path exist, there is ample resources for inode and/or disk block allocation and appropriate permissions along all directory components of the path are available.

Parameters
pathThe path at which the file must be created.
permissionsThe permissions to associate with the file.
overwriteIf true, recreates the file, effectively truncating previous content.
Exceptions
{std::system_error}reason for failure of operation.
