Detailed Description

Defines a structure for abstracting I/O operations via system calls.

File objects work over a file descriptor referencing a global file table entry maintaining I/O mode and offset. They follow the RAII idiom and perform auto-closing of the file table entry when the corresponding object runs out of scope. Descriptor ownership can however be transferred to other objects through move construction or move assignment.
